# SafeScan Mobile Typography Plan

The website uses a two-font system that is already available in the app:

- Main titles: `Orbitron-Bold` or `Orbitron-Black`, uppercase only when the label is a system/verdict label. Use 28-42px on mobile screens.
- Subtitles and section headings: `Orbitron-Bold` for short labels, `SpaceGrotesk-SemiBold` for human-readable headings like tier names.
- Body text: `SpaceGrotesk-Regular`, 14-16px with 22-24px line height.
- Metadata, URLs, codes, and counts: `SpaceMono-Regular`, 12-18px depending on density.
- Eyebrows: `Orbitron-Bold`, 11-12px, uppercase, accent color, wide tracking.
- Buttons and badges: `Orbitron-Bold`, 12-15px, 8px radius for rectangular controls and pill radius only for badges.

Mobile screens should follow the website hierarchy:

1. Accent eyebrow.
2. Display-font page title.
3. Muted body-font support copy.
4. Dense bordered panels with display-font section labels.
5. Mono text only for machine-like values such as scan counts, links, wallet addresses, and QR payloads.
